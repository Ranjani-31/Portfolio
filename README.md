# 🌐  Portfolio

A personal portfolio website to showcase my skills, projects, experience, and contact information. Built with a focus on clean design, responsiveness, performance, and accessibility.

## 📌 Overview

This portfolio serves as a digital resume and personal brand platform. It highlights my technical skills, real-world projects, and professional journey as a developer.

The goal is to present information clearly while maintaining a modern and minimal user experience.

## ✨ Features

- Responsive design (mobile, tablet, desktop)

- About me section

- Skills & technology stack

- Projects showcase

- Resume download

- Smooth navigation and animations

- SEO-friendly structure
